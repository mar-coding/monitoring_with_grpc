# monitoring_with_grpc
monitor the server with the help of python3 and grpc protocol

## run(both in client and server)
```
python3 -m venv venv
source venv/bin/activate
sudo chmod +x ./setup.sh
pip install -r req.txt
./setup.sh
```

### for server
```
python server.py
```

### for monitoring
```
python monitor.py
```
